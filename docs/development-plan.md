# 步步兽开发路线

更新日期：2026-07-14

## 已完成

1. Milestone 0–1：Electron 框架、透明置顶窗口、Sprout Cat 动画与点击/拖动交互。
2. Milestone 2–4：SQLite 任务、今日计划、可恢复专注、XP 与等级成长。
3. Milestone 5：Hermes 健康检查、任务拆解、每日计划、只读对话与 AI COO。
4. Milestone 6：Obsidian 只读项目解析、确认式项目同步与确认式每日复盘写入。
5. Milestone 7–8 基础版：项目视图、提醒设置、opt-in 应用使用统计与 Windows 安装包。
6. UI 迭代：暖色今日插件、日常/沉淀/系统分组工作台、全局快捷操作与多显示器窗口边界适配。

## 当前验证目标

- 连续 7 天真实使用，验证桌宠是否降低开始任务的阻力。
- 检查晨间规划、专注、完成奖励、晚间复盘是否形成闭环。
- 记录误提醒、Hermes 建议无效、项目同步遗漏和复盘不准确的真实案例。

## 下一批优先级

1. P0：修复真实使用中造成数据错误、窗口无法使用或复盘误写的问题。
2. P1：增加应用图标、发布说明和 Windows 代码签名。
3. P1：为 Hermes/Obsidian 增加面向非技术用户的安全配置引导。
4. P2：周复盘历史趋势和更细的项目健康度，不增加更强的电脑监控。

## 外部条件

- 自动更新需要 GitHub Release 发布地址和发布凭据。
- 消除 Windows SmartScreen 信誉提示需要代码签名证书。
- 这些凭据只能配置在本机或 GitHub Secrets，禁止写入仓库。
